echo "# 🚀 Flancer — Freelance Services for Everyone

Flancer is a modern freelance marketplace platform that allows beginner freelancers to offer affordable services and users to find talents that match their budget and project needs. Built with **Next.js**, **Supabase**, and a focus on clean UX, Flancer empowers accessible digital work for all.

## 🌟 Features

- 📦 Upload freelance services with image, description, category, pricing, and delivery time.
- 🔎 Browse available services with filtering (by category, location, price).
- 💼 Personalized dashboard to view and manage posted services.
- 📸 Image uploads stored securely via **Supabase Storage**.
- 🧠 Smooth user experience using **ShadCN UI** components and responsive design.

---

## 🛠️ Tech Stack

- **Frontend**: Next.js, React, TailwindCSS, ShadCN UI  
- **Backend/Database**: Supabase (Auth, DB, Storage)  
- **State Management**: React Hooks  
- **Deployment**: Vercel

## 🚧 Setup Instructions

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/your-username/flancer.git
   cd flancer
   \`\`\`

2. **Install dependencies**
   \`\`\`bash
   npm install
   \`\`\`

3. **Set up environment variables**
   Create a \`.env.local\` file in the root directory with the following:
   \`\`\`
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   \`\`\`

4. **Run the development server**
   \`\`\`bash
   npm run dev
   \`\`\`
